WHATFIX 벤치마킹 포인트

① 실시간 학습 개입 모델

현황 → 기회

기존 기업교육은 '교육 → 현장 적용'이 단절되어 있다. Whatfix 모델은 업무 수행 중 AI가 실시간으로 개입해 즉시 적용 가능한 가이던스를 제공한다.

적용: AI 시뮬레이션 세션 중 학습자 막힘 구간 자동 감지 → 힌트 삽입 로직 설계 참고

② 마찰 포인트 데이터 분석

현황 → 기회

어떤 단계에서 얼마나 많은 학습자가 이탈하는지를 데이터로 측정한다. 이는 교육 설계 개선의 근거가 된다.

적용: 시나리오별 완료율·이탈 포인트 분석 대시보드 설계 시 지표 체계 참고

③ 콘텐츠 제작 자동화

현황 → 기회

Authoring Agent가 가이던스 초안을 자동으로 생성하는 방식은 기업 맞춤 교육 자료 제작에 응용 가능하다.

적용: 기업별 시나리오 초안 AI 생성 → 검수·보완 워크플로우 효율화

① 콘텐츠 자산 진단 서비스화

현황 → 기회

"현재 어떤 역량이 커버되고 어떤 역량이 빠져 있는가"를 AI가 진단한다. 이는 교육 컨설팅에서 강력한 진입 도구가 된다.

적용: 고객사 기존 교육 콘텐츠를 받아 역량 갭 진단 → 교육 필요성 근거 제시

현황 → 기회

적용: 기존 교육 자산 보유 기업 대상 콘텐츠 리뉴얼·마이그레이션 제안

7-1 · 기술 연동 상세
Whatfix — LMS·시스템 연동
연동 대상방법난이도기간
SAP SuccessFactors브라우저 확장 자동 감지낮음1~3일
SalesforceChrome Extension + AppExchange낮음1~5일
Workday브라우저 확장 (iframe 이슈 주의)중간3~7일
Microsoft 365브라우저 기반 자동 적용낮음1~2일
커스텀 LMSAPI + Embed 코드 삽입중간1~2주
VDI / Citrix전용 에이전트 설치 필요높음2~4주
MetaLark.AI — LMS·스토리지 연동
연동 대상방법난이도기간
SCORM 패키지파일 업로드 또는 API낮음즉시
Google DriveOAuth 연동낮음30분
SharePointOAuth + API낮음1~2시간
AWS S3IAM 키 기반중간2~4시간
Oracle LMS전용 커넥터중간1~3일
국내 LMS (자체)Open API/Swagger 표준중간1~2주

핵심: Whatfix는 애플리케이션 코드 수정 없이 브라우저 확장 레이어로 작동 — IT 개발 리소스 불필요. MetaLark는 Open API 표준으로 대부분의 시스템과 연동 가능.

7-2 · 콘텐츠 제작·활용 워크플로우
Whatfix — 실무자 6단계
1
가이던스 프로젝트 생성 (5분)

Whatfix 콘솔 접속 → 대상 애플리케이션 선택 → 프로젝트 생성

2
Authoring Agent로 초안 생성 (10~30분)

텍스트 프롬프트 입력 → AI 초안 자동 생성 → 구조·순서 표준화

3
검토·수정 (30분~2시간)

스텝별 텍스트·UI 캡처 확인 → 스텝 추가·삭제·순서 변경

4
미리보기·QA (30분)

실제 앱에서 가이던스 작동 확인 → 모바일·데스크탑 테스트

5
배포 (5분)

대상 사용자 그룹 선택 → 트리거 조건 설정 (첫 로그인·특정 페이지 등)

6
성과 모니터링 (주 1회)

Insights Agent 대시보드에서 완료율·이탈 포인트 확인 → 이탈 구간 수정

가이던스 1개 기준: 수동 4~8시간 → Whatfix 1~2시간 (~75% 단축)
MetaLark.AI — 실무자 5단계
1
콘텐츠 라이브러리 연결 (1~4시간)

SharePoint·Google Drive·S3 OAuth 연동 또는 파일 직접 업로드

2
자동 분석 실행 (30분~수 시간)

Parsing Engine 전 콘텐츠 구조화 → Skills Engine 자동 태깅 → 분석 리포트 생성

3
갭 분석 리포트 확인 (1~2시간)

역량 커버리지 시각화 → 구식·중복 콘텐츠 목록 확인

4
메타데이터 정비 (배치 처리)

AI 생성 메타데이터 검토 → 승인 또는 수정 → LMS 업로드용 CSV 내보내기

5
마이그레이션 실행 (필요 시)

대상 LMS 연동 → 정비된 콘텐츠 일괄 이전

콘텐츠 100개 감사 기준: 수동 5~10인일 → MetaLark 0.5~1인일 (~90% 단축)
